## AS10099 是什么？为什么联通用户要盯着它

说白了，联通的网络分三层：

**第一层：AS4837（联通 169 网）**
这是绝大多数普通家宽走的线路。便宜，量大，缺点是晚高峰堵死，丢包率飙升，体验全靠运气。买了没做优化的 VPS，大概率就是这个。

**第二层：AS9929（联通 A 网 / 联通 CNCNET）**
企业宽带和 IDC 常用的线路，避开了最拥堵的普通 169 网，延迟和丢包率好不少。但现在已经越来越难算是"精品"了，更多时候只是比 169 网好一点。

**第三层：AS10099（中国联通国际 / China Unicom Global / CUP）**
这才是联通高端产品线的旗舰。全称是 China Unicom Global，负责联通的国际精品网络。国内段走 AS9929 入境，国际段全程 AS10099。延迟低、丢包少、晚高峰稳——这才是联通用户真正值得为之掏钱的线路。

有时候你会看到 AS10099 被叫做 "CUP"（China Unicom Premium），两个叫法说的是同一条线。

所以，如果你是联通用户，在选 VPS 时看到 AS10099，那就是直接对号入座了。
